# Titanic Survival Prediction 🚢

## 📌 Project Overview

This project predicts whether a passenger survived or not using Machine Learning (Logistic Regression).

---

## 👥 Team Members & Roles

* **Member 1 (Data Loading)**
  Loaded Titanic dataset using pandas.

* **Member 2 (Feature Engineering)**
  Converted categorical data (Sex) into numeric and handled missing values.

* **Member 3 (Data Analysis)**
  Displayed dataset info, head, and statistics.

* **Member 4 (Model Building)**
  Trained Logistic Regression model.

---

## ⚙️ Workflow

1. Load data from CSV files
2. Clean and preprocess data
3. Analyze dataset
4. Train machine learning model
5. Display results

---

## 📂 Files Description

* `main.py` → Main file (runs full project)
* `member1_data.py` → Data loading
* `member2_feature_engineering.py` → Data preprocessing
* `member3_analysis.py` → Data analysis
* `member4_model.py` → Model training
* `Titanic_train.csv` → Training dataset
* `Titanic_test.csv` → Testing dataset

---

## ▶️ How to Run

```bash
python main.py
```
